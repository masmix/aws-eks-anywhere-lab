# Installation manual

# Prerequisities

- aws cli installed on your local machine 

## AWS configure CLI

Run following

```sh
aws configure 
```

or 

```sh
aws configure sso
```

depends on your AWS environment.


## Execute IAM script

```sh
bash configure-iam.sh 
```

After corect configuration

[Go back to main README.md](scripts/aws/README.md.md)

to continue.
